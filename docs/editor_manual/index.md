# Editor's guide

Wagtail’s Editor Guide now has its own website: guide.wagtail.org.

This guide is written for users of a Wagtail-powered site, including content editors, moderators, and administrators who manage content on a day-to-day basis through the Wagtail admin interface.

After logging into the admin panel, users can create and edit pages, upload media, and manage site content without needing technical knowledge.

For detailed guidance on using the editor interface, visit guide.wagtail.org.
